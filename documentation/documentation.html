Requirements

For installing this template you need to make sure that you have the following extensions on your webserver. You will need to contact your webhost if the following requirements are missing:

  • PHP 5.2+;
  • Apache webserver (For the Page Builder Windows servers will probably work, but are not supported)
  • PHP Mail support;
  • Enabling JavaScript in the web browser.

To be able to edit and upload the theme you need also:

  • A text editor like Sublime Text 2, Notepad++ or Brackets
  • An FTP tool to upload the files like Filezilla or Cyberduck

Installation

Unzip the downloaded files to the local disk on your computer. Inside the extracted folder there will be a folder with documentation and a folder with the html templates. Choose pages which you will use in project and copy all the content related to the layout that interests you to your server via FTP client.

In this archive some images that you can see in demo pages were replaced by a placeholder.

Structure

As Kounter is an HTML template, it uses a number of .html pages in conjunction with css stylesheets and a number of javascript plugins. The following is a brief overview of the files and a look at some prominent features.

We do our best while making all files for Kounter. Every file (HTML, CSS, JS) that was created by us has clear code, wrapped with comments – all this for your convenience.

Each demo has similar file structure:

  • css – CSS Styles Folder
  • images – Image Folder
  • video – Video Folder
  • fonts – Fonts Folder
  • js – Javascript Files Folder
  • php – PHP Form Folder
  • index.html – Main HTML File

Fonts

Kounter uses "Source Sans Pro" Google WebFont. You can choose other fonts from google directory editing this line of code in the <head>.

<!-- Google Fonts -->
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Source+Sans+Pro:300,400,600,700">

If you want to import new font from Google Webfont library, don't forget to change property in CSS style: font-family with correct font name.

<!-- css/style.css -->

body {
    font-family: "Source Sans Pro", Helvetica, Arial, sans-serif;
}

Kounter also uses Montserrat Font. If you wanna use another or more font faces of this font you can download them on Github and include the font face in the style.css (/css).

Count Down

The Kounter Template uses the countdown-Plugin to show animated countdown. The countdown could be defined through the attribute data-end-date. To change the date, please use the attribute data-end-date (index.html).

  • data-end-date: The number to start counting down.
<!—- HTML (index.html) -—>
<div class="count-down p-t-b-15" data-end-date="Jan 1, 2018 00:00:00">
    ...
</div>

// Javascript (custom.js)
var time = $('.count-down'),
  endDate = time.data("end-date");
  time.countdown({
    date: endDate,
    render: function (data) {
      $(this.el).html('<div><h1>'
        + this.leadingZeros(data.days, 3)
        + '</h1><p>days</p></div><div><h1>'
        + this.leadingZeros(data.hours, 2)
        + '</h1><p>hrs</p></div><div><h1>'
        + this.leadingZeros(data.min, 2)
        + '</h1><p>min</p></div><div><h1>'
        + this.leadingZeros(data.sec, 2)
        + '</h1><p>sec</p></div>');
  }
});

Page Backgrounds

Image Background

To change the background image in the header you need to go to the style.css, scroll down to „08. Backgrounds“ and change the link of the image to the name you stated for your background image. Please make that you copy your image into the images folder.

Solid Background

To change the background color in the header you need to go to the style.css, scroll down to „03. Backrounds“ and change the colors of the class ".bg-solid-color" to the colors of your needs.

Youtube Background

Kounter provides the opportunity to embed a youtube-video in background.

Youtube video embedding accomplished through the attribute data-video-id. You only need to provide the video ID which you can find in the url of the video:

https://www.youtube.com/watch?v=8-vvXLaozRo

Youtube video is only enabled for Desktop – videos for mobile and tablets are disabled. Only fullscreen poster will be used.

Use the img-tag to change the name of the preview image of the youtube-video.

<div id="page-youtube" data-video-id="8-vvXLaozRo">
    <img src="images/page-youtube-image.jpg" alt="">
</div>
The Youtube Video is only enabled for Desktop - videos for mobile and tablets are disabled. Only fullscreen poster will be used. Use the img-tag to change the name of the preview image of the video.

Video Background

Kounter provides the opportunity to add a video in background (which should be composed of a screenshot jpg, a file .mp4, a file .ogv and a file .webm).

To change the file name and the directory where the video is please use the attributes data-path und data-file.

Use the img-tag to change the name of the preview image of the video.

<div id="page-video" data-path="video/" data-file="video">
    <img src="video/video.jpg" alt="">
</div>
Video is only enabled for Desktop - videos for mobile and tablets are disabled. Only fullscreen poster will be used. Use the img-tag to change the name of the preview image of the video.

Rain Background

Kounter provides the opportunity to add a particle background. To adjust the rain drops to your needs please use the documentation which is provided online at Github

To edit the rain drops please open the custom.js (/js), change sizes, quantity etc. Use the img-tag to change the name of the image.

                  // HTML
<div id="page-rain" class="section-overlay media">
  <img id="page-rain" alt="" src="images/placeholder-2400-1600.jpg">
</div>

// Javascript (js/custom.js)
var pageRain = $('#page-rain');
if (pageRain.length) {
  var image = pageRain[0];
  image.onload = function () {
    var engine = new RainyDay({
      image: this,
      parentElement: $('.section-overlay')[0]
    });
    engine.rain([[1, 2, 4000]]); // add 4000 drops of size from 1 - 2
    engine.rain(
      [
        [3, 3, 1], [5, 5, 1], [6, 2, 1]
      ],
    100); // every 100ms
  };
  image.crossOrigin = 'anonymous';
  image.src = pageRain.attr('src');
}

Ripple Background

Kounter provides the opportunity to add a ripple / drops background. To adjust the ripple effect to your needs please use the documentation which is provided online at Github

To edit the ripple please open the custom.js (/js), change resolution, dropRadius etc.

                  // HTML
<div id="page-ripple" class="section-overlay page-image-ripple">
  <canvas></canvas>
</div>

// Javascript (js/custom.js)
var ripple = $('#page-ripple');

if (ripple.length) {
  ripple.ripples({
    resolution: 512,
    dropRadius: 10, //px
    perturbance: 0.04,
    interactive: false
  });

  setInterval(function () {
    var $el = ripple;
    var x = Math.random() * $el.outerWidth();
    var y = Math.random() * $el.outerHeight();
    var dropRadius = 20;
    var strength = 0.04 + Math.random() * 0.04;

    $el.ripples('drop', x, y, dropRadius, strength);
  }, 3000);
}

  1. Login to your mailchimp account
  2. Go to your list
  3. Select Signup Forms
  4. Select Embedded forms

Contact Form

To connect the contact form to your email account please enter your email address into the contact-process.php under: $emailTo = ' ';

Additionally you are able to change the subject line of the email you will get after sending a message for this please use the php line: $subjectPrefix = ' ';

To make bigger changes to the included forms you will need professional skills in PHP and Javascript. Please be careful.

Further adjustments (e.g. adding an additional form field) can be made in the HTML form, contact-form.js und contact-process.php

To change the highlighting text colors you need to open the style.css (/css). Scroll down to the property ".color" of the ".error .form-control:-moz-placeholder"-, ".error .form-control::-moz-placeholder"-, ".error .form-control:-ms-input-placeholder"-, ".error .form-control::-webkit-input-placeholder"-selectors and change the color to what you need. For the message container you need to change the color-property of ".message"- and ".error .message"-selector
